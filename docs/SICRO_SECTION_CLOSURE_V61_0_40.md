# SICRO Section Closure — v61.0.40

A v61.0.40 mantém a regra de coleção:

- SICRO com item → `principais`
- SICRO sem item → `auxiliares_globais`

Além disso, audita seções A-F:

- A: Equipamentos
- B: Mão de obra
- C: Materiais
- D: Atividades auxiliares
- E: Tempo fixo
- F: Momento de transporte

Campos ausentes nessas seções entram no correction document.
